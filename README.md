# broc-launch

![A screenshot of the application](./screenshot.png)

A GTK4 application to quicken web search and AI questions.

## Compiling, Running

### Compiling

Requires Python, `dbus-python`, and `gtk4-layer-shell` to be installed systemwide.

### Running

> python -m broc_launch.main

from the root directory.

## Installing

Run the script `reinstall.sh`.
